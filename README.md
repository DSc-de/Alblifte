# Alblifte

Öffentliche Datenbank zu den Skilifte auf der Schwäbischen Alb. Alle 67 Lifte / Skigebiete auf der Alb sind enthalten.

Korrekturen und Fehlermeldungen: Am einfachsten direkt als [GitHub Issue](https://github.com/DSc-de/Alblifte/issues/new/choose)

## Format
Derzeit steht die Sammlung als [SQLite Datenbank](https://github.com/DSc-de/Alblifte/blob/main/data/Alblifte.sqlite) zur Verfügung. 

## Demo
Die [Website der Alblifte Datenbank](https://dsc-de.github.io/Alblifte/) listet alle Datensätze.

Die Alblifte Datenbank wird verwendet von [Alblifte.de](http://www.alblifte.de).

Enthalten ist eine einfache [Python Demo](https://github.com/DSc-de/Alblifte/blob/main/demo/SqliteDemo.py) die den Zugriff auf die SQLite Datenbank zeigt.

* Name
* Website
* Position (Breite, Länge)
* Status Link
* Webcam Link
* Schnee-Telefon
* Kann gemietet werden
